Welcome to Sample E-commerce

Sample E-commerce is a take on a simple e-commerce website that shows items the client side window. It features different functions such as adding to cart as well as cart checkouts. Another thing to note is that this website can be viewed easily in any device. Listed below are some questions you might have as well as the tools used to create this project.

**Note: All products and services in this project are not real and are used for demonstration purposes only.

I. How to use the website

How to add a product to your cart

  1. 1. Choose a Product that you like from the Homepage.
  2. 2. Once selected, type in the quantity that you want of this product. It is defaulted to one.
  3. 3. Press the "add to cart" button.
  4. 4. You may now see your products in the cart page.

How to modify your cart

  1. 1. In your carts page, select a product that you want to edit.
  2. 2. Once selected, you can either change the quantity or remove the product from your cart.
  3. 3. After selecting a button, you will then be redirected back to the cart page with your updated products.

How to checkout your cart

  1. 1. After finalizing the products that you want, fill out the forms with your name, cellphone number, and home address.
  2. 2. Click the checkout button at the bottom of the form to submit your order.
  3. 3. A notification for a successful checkout will appear.
  4. 4. Congratulations! You have successfully completed your first checkout.

II. Tools used to create the website

The following are the different tools and technologies that were used in the creation of this project.

  • Angular 17 - This project is built using the latest version of the Angular Framework. This is a Type-script based framework developed by Google.
  • Tailwind CSS - Tailwind CSS is used not only to create the style of the website but to also reduce development time. It is also used to integrate responsive UI design into the project.
  • Fakestore API - The products shown in this project is from an open-source API called Fakestore. It is used to get the products and product categories from an online source.
  • FontAwesome icons - Icons shown in this project are taken from the FontAwesome Website.
  • Angular Material - Used to create pre-built components developed by the Angular Team. In this particular project, this was used to create the checkout dialog.
  • Ng-angular-popup package - Used to show toast notifications when user completes an action.
  • Ngrx - Ngrx is used to manage different states within the application such as the cart products.
